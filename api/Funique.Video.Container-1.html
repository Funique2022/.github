Class Container<T>

File or playalibe object information list
This extends the action of list class
When push into the list it will trying to generate instance of object and assign reference to it
When remove the object in list, It trying destroy object for it
User can inherit the class to override Add(T t), Remove(T t) to extends to action
------------------------------------------------
檔案 或是 可被播放物件 的 資訊清單
延伸 List 類別的動作
當加入新的物件, 嘗試生成 參考物件 並且 賦值
刪除時將場景物件直接銷毀
用戶能嘗試繼承此類別來延伸更多動作
