## Github Action

[學習文件](https://docs.github.com/en/actions/creating-actions/creating-a-docker-container-action)

透過 Github Action 將代碼的合併, 更新綁定事件. 事件可以定義流程像是測試, 建置, 部屬. 可以與 AWS Lightsail 實體上架設 CI/DI 服務進行自動部屬, 自動整合.

> [!WARNING]
> 這將會是一個龐大的工程...
