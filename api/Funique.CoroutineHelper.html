Class CoroutineHelper

Library code does not inherits UnityEngine.MonoBehaviour
Which means these class does not have way to access UnityEngine.Coroutine feature that unity provide
To improve this problem, We add a singleton monoBehaviour script to do all UnityEngine.Coroutine work
