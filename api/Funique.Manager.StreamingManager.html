Class StreamingManager

Handle all things related to streaming services
VOD and Live and Casting and None-Casting
And handle the interaction between UI component and other manager

Casting mode will transcoding the HLS streaming to master server local directroy
Then user can get the streaming from nginx services

The purposes of this is to save network usages
None-Casting mode will send the URL to user, telling them to get video from online
------------------------------------------------
處裡所有跟串流服務相關的事務
VOD 和 Live and 轉介 和 非轉介
還有處理與使用者介面 跟 其他元件的互動

轉介模式會將 HLS 串流存在 伺服器的本地資料夾
然後用戶再透過 Nginx 撈

目的是為了節省網路流量
非轉介模式會直接命令用戶播放 URL, 用戶會直接占用頻寬
